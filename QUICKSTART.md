## Step 1: Download

Download the latest version from the [Releases](https://github.com/yourusername/starkblock/releases) page or clone this repository.

```bash
git clone https://github.com/yourusername/starkblock.git
```

## Step 2: Install in Chrome

1. Open Chrome and navigate to: `chrome://extensions/`
2. Enable **Developer Mode** (toggle in top-right corner)
3. Click **"Load unpacked"**
4. Select the `starkblock` folder
5. StarkBlock will appear in your extensions!

## Step 3: Pin the Extension

1. Click the **Extensions** icon (puzzle piece) in Chrome toolbar
2. Find **StarkBlock** in the list
3. Click the **pin** icon to keep it visible

## Step 4: Start Browsing!

That's it! StarkBlock is now protecting you. The arc reactor icon will glow blue when active.

## Troubleshooting

### Extension not showing up?
- Make sure Developer Mode is ON
- Try reloading the extension
- Check browser console for errors

### Site not loading properly?
- Try switching to Stealth Mode
- Add the site to whitelist if you trust it
- Report the issue on GitHub

### Stats not updating?
- Refresh the popup
- Check if protection is enabled (toggle should be blue)
- Clear browser cache and reload
